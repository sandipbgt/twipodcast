Welcome to twipodcast
============================

`twipodcast <http://sandipbgt.github.io/twipodcast>`_ is a web app that allows
you to listen podcast or any mp3 media file on your phone for free without **internet**.

The webapp is live at `http://sandipbgt.github.io/twipodcast <http://sandipbgt.github.io/twipodcast>`_

The api is live at `https://twipodcast.herokuapp.com <https://twipodcast.herokuapp.com>`_

The api is written in `Flask <http://flask.pocoo.org>`_ framework, frontend app
is written in `AngularJS <http://angularjs.org>`_ framework and `Twilio <http://twilio.com>`_
calls api is used for phone calls.

The source code for api server and frontend app is open source and `available on github <https://github.com/sandipbgt/twipodcast>`_.
